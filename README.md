###How To Run
1. "python3 -m venv .venv"
2. ". .venv/bin/activate"
3. "pip install -r requirements.txt"
4. "flask --app finance_tracker initdb"
5. "flask --app finance_tracker run"

OR 

Use the makefile: 
1. "python3 -m venv .venv"
2. ". .venv/bin/activate"
2. "make init"
3. "make run"
